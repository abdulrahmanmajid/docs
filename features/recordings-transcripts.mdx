## Integration Features

<CodeGroup>
  <Code title="CRM Integration">
  ```typescript
  // Automatically attach to CRM
  onCallComplete: async (call) => {
    await crm.attachRecording({
      customerId: call.customerId,
      recordingUrl: call.recording,
      transcript: call.transcript,
      summary: call.aiSummary
    });
  }
  ```
  </Code>
  
  <Code title="Analytics Export">
  ```typescript
  // Export call analytics
  exportAnalytics: async (dateRange) => {
    const report = await analytics.generate({
      startDate: dateRange.start,
      endDate: dateRange.end,
      metrics: ['sentiment', 'keywords', 'outcomes'],
      format: 'detailed'
    });
    return report;
  }
  ```
  </Code>
</CodeGroup>
