# Outbound Calling Campaigns

Create and manage automated outbound calling campaigns that scale with your business needs.

## Getting Started

<Note>
Before starting an outbound campaign, make sure you have:
- A verified caller ID
- Your contact list ready
- Agent prompt configured
- Required variables defined
</Note>

## Setting Up a Campaign

<Steps>
  <Step title="Create Campaign">
    Set your campaign basics:
    ```typescript
    Campaign Name: "January Appointment Reminders"
    Type: Batch Campaign
    Timezone: UTC-5 (Eastern)
    Daily Call Window: 9 AM - 5 PM
    ```
  </Step>
  
  <Step title="Configure Agent">
    Set up your AI agent's behavior:
    ```typescript
    Agent Name: "Sarah"
    Language: English (US)
    Voice Type: Female, Natural
    Speaking Rate: 1.0x
    ```
  </Step>
  
  <Step title="Upload Contacts">
    Prepare your contact list CSV:
    ```csv
    phone,first_name,appointment_date,service_type
    +1234567890,John,2024-01-15,Dental Cleaning
    +0987654321,Sarah,2024-01-16,Check-up
    ```
  </Step>
  
  <Step title="Set Call Rules">
    Define your calling rules:
    - Max attempts: 3
    - Retry interval: 30 minutes
    - Call timeout: 60 seconds
    - Concurrent calls: 10
  </Step>
</Steps>
