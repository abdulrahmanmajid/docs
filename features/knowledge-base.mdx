# Knowledge Base Management

Empower your AI agents with custom knowledge. Create, organize, and manage information that your agents can use during calls.

<Note>

## Training Your Agent

<Steps>
  <Step title="Add Base Knowledge">
    Start with essentials:
    ```typescript
    baseKnowledge: {
      company: "Basic company info",
      products: "Product catalog",
      services: "Service details",
      policies: "Company policies"
    }
    ```
  </Step>
  
  <Step title="Define Responses">
    Create response templates:
    ```typescript
    responses: {
      greeting: "Welcome to {{company.name}}",
      productInfo: "{{product.name}} costs {{product.price}}",
      support: "Let me help you with {{issue.type}}"
    }
    ```
  </Step>
  
  <Step title="Test Understanding">
    Verify agent comprehension:
    - Test questions
    - Role-play scenarios
    - Edge cases
    - Complex queries
  </Step>
</Steps>

## Integration Options

<CodeGroup>
  <Code title="API Integration">
  ```typescript
  // Fetch knowledge dynamically
  async function getKnowledge(query) {
    const response = await kb.search({
      query: query,
      context: currentCall.context,
      confidence: 0.8
    });
    return response.bestMatch;
  }
  ```
  </Code>
  
  <Code title="Real-time Updates">
  ```typescript
  // Update knowledge during call
  onNewInformation: async (info) => {
    await kb.update({
      category: info.category,
      content: info.content,
      priority: "high",
      source: "call-learning"
    });
  }
  ```
  </Code>
</CodeGroup>
