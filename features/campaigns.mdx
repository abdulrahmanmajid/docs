# Campaign Management

Create, schedule, and monitor your outbound calling campaigns. Reach thousands of customers with personalized AI conversations.

## Creating a Campaign

<Steps>
  <Step title="Basic Setup">
    Start with campaign essentials:
    ```typescript
    Campaign Details:
    - Name: "Holiday Special"
    - Type: Outbound
    - Priority: High
    - Description: "Holiday promotion calls"
    ```
  </Step>
  
  <Step title="Target Audience">
    Select your contacts:
    - Upload contact list
    - Choose existing list
    - Filter contacts
    - Set priority order
  </Step>
  
  <Step title="Schedule">
    Set your campaign timing:
    ```typescript
    Schedule:
    - Start Date: "2024-01-15"
    - End Date: "2024-01-30"
    - Call Window: "9 AM - 6 PM"
    - Time Zone: "Customer Local"
    ```
  </Step>
</Steps>

## Agent Configuration

<Steps>
  <Step title="Voice Settings">
    Customize your agent's voice:
    ```typescript
    voice: {
      type: "female",
      accent: "neutral",
      speed: 1.0,
      pitch: "natural"
    }
    ```
  </Step>
  
  <Step title="Script Setup">
    Define conversation flow:
    ```typescript
    script: {
      greeting: "Hello {first_name}, this is {agent_name}...",
      mainMessage: "I'm calling about {campaign.purpose}...",
      closing: "Thank you for your time..."
    }
    ```
  </Step>
  
  <Step title="Response Handling">
    Configure how your agent handles:
    - Customer questions
    - Objections
    - Call backs
    - Do not call requests
  </Step>
</Steps>

## Integration Options

<CodeGroup>
  <Code title="CRM Integration">
  ```typescript
  // Update CRM after each call
  onCallComplete: async (result) => {
    await crm.update({
      contactId: result.contactId,
      status: result.outcome,
      notes: result.summary,
      nextAction: result.followUp
    });
  }
  ```
  </Code>
  
  <Code title="Webhook Notifications">
  ```typescript
  // Send webhook for important events
  onCampaignEvent: async (event) => {
    await webhook.send({
      type: event.type,
      campaign: event.campaignId,
      data: event.details,
      timestamp: event.time
    });
  }
  ```
  </Code>
</CodeGroup>
