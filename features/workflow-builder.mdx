## Creating Workflows

<Steps>
  <Step title="Start New Workflow">
    Begin with a trigger:
    ```typescript
    Trigger Types:
    - Inbound Call
    - Outbound Campaign
    - Schedule
    - API Event
    ```
  </Step>
  
  <Step title="Add Steps">
    Drag and drop components:
    - Voice interactions
    - Decision points
    - Actions
    - Integrations
  </Step>
  
  <Step title="Configure Logic">
    Set up conditions:
    ```typescript
    if (userInput === "sales") {
      route.to("sales-department")
    } else if (timeOfDay > "17:00") {
      playMessage("after-hours")
    }
    ```
  </Step>
  
  <Step title="Test and Deploy">
    - Preview flow
    - Test scenarios
    - Debug issues
    - Go live
  </Step>
</Steps>

## Integration Examples

<CodeGroup>
  <Code title="CRM Update">
  ```typescript
  // Update CRM after call
  onCallEnd: async (callData) => {
    await crm.update({
      customerId: callData.customer.id,
      interaction: {
        type: 'call',
        duration: callData.duration,
        outcome: callData.resolution
      }
    })
  }
  ```
  </Code>
  
  <Code title="API Webhook">
  ```typescript
  // Send webhook on specific event
  onTransfer: async (callInfo) => {
    await webhook.send({
      event: 'call_transfer',
      department: callInfo.targetDept,
      reason: callInfo.transferReason
    })
  }
  ```
  </Code>
</CodeGroup>
